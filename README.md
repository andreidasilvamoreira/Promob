# Promob

Backend para geração de **orçamentos de móveis planejados**.

O objetivo do projeto é criar um sistema que permita marceneiros cadastrar materiais, módulos de móveis e gerar **estimativas de custo e preço** para projetos de clientes.

## 🚀 Tecnologias

- **NestJS**
- **TypeScript**
- **PostgreSQL**
- **SQL manual (sem ORM)**
- **Node.js**

## 📦 Funcionalidades (V1)

- Cadastro de usuários
- Cadastro de clientes
- Cadastro de materiais
- Cadastro de módulos de móveis (products)
- Definição de peças dos módulos (product parts)
- Criação de projetos de clientes
- Adição de módulos aos projetos
- Geração de estimativas de orçamento
